## LAMBDA FILE SYSTEM PROJECT

### Introduction
This is a Java Spring boot project built with Gradle. The project is built
with [Spring Reactor]("https://projectreactor.io/docs/core/release/reference/#intro-reactive") methodology.


### Technologies
* Redis
* RabbitMQ
* MySQL
